Welcome to the Labyrinth Game

Start Game

Star game: Click on Start Game link.
Controls: Forward: W key, Backward: S key, Left: A key, Right: D key.
Direction: Click on the game site and move the mouse for direction.
To go to the map section: CTRL and arrow up.
To rotate the block in map section: arrow left and arrow right.
To slide the blocks: Enter key.
To go to the 3D game again: CTRL and arrow down.
